# dotfiles

My dotfiles are based on [mathiasbynens/dotfiles](https://github.com/mathiasbynens/dotfiles), feel free to fork or reuse these dotfiles.

## Usgae

Clone this repo, modify `.gitconfig`:

```
[user]

	name = your_name
	email = your_email
```

Then run `source bootstrap.sh`.
