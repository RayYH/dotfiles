# Dotfiles

> [!CAUTION]
> DO NOT BLINDLY USE MY DOTFILES. I STRONGLY RECOMMEND YOU TO READ THROUGH THE FILES AND UNDERSTAND WHAT YOU ARE DOING.

## Installation

```bash
git clone https://github.com/rayyh/dotfiles
cd dotfiles
yes | ./bootstrap.sh
./install.sh
```

Reload your shell and you are good to go.

## Supported Platforms

- macOS
- Ubuntu
- Arch Linux
